# 0007 — Share button not functional on post cells

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The share (box-with-arrow) button below each post in the feed has no effect when tapped or clicked.

## Steps to reproduce

1. Launch the app and sign in.
2. Navigate to any feed.
3. Tap or click the share button on any post.

## Expected behavior

The system share sheet (`ShareLink` / `UIActivityViewController` / `NSSharingServicePicker`) opens with the post's public URL (`https://bsky.app/profile/<handle>/post/<rkey>`) ready to share.

## Actual behavior

Nothing happens. The system share sheet does not appear.

## Attachments

- `issues/0004/screenshot.png` — post action bar (shared screenshot)

## Notes

On macOS use `NSSharingServicePicker`; on iOS use `ShareLink` or `UIActivityViewController`. Confirm the post's public bsky.app URL is being constructed correctly from the post's `uri` (AT URI → did + rkey → handle + rkey lookup or direct did-based URL).
