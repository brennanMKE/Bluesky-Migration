## Description

Manual sign-off for the moderation suite — blocks, mutes, mute lists,
content labelers, the report flow, and post-interaction settings.

## Test sequence

1. **Open Moderation.** From Profile → `…` → Moderation. Confirm
   the landing screen has sections for: Muted users, Blocked users,
   Mute lists, Block lists, Content labelers, Content filters.
2. **Muted users list.** Confirm a search field at the top of the
   list (#0134). Confirm muted users render as rows with avatar /
   name / "Unmute" button.
3. **Blocked users list.** Same shape (#0134).
4. **Unmute / unblock.** Tap "Unmute" on a row. Confirm the row
   disappears immediately (optimistic) and stays gone after a
   refresh.
5. **Mute / block from feed.** Open the `…` menu on a post from a
   user not in either list. Tap Mute. Confirm a confirmation sheet
   (matches RN). Confirm. Confirm the user appears in the Muted
   list and their posts disappear from the feed on next refresh.
6. **Mute lists.** Open Mute lists. Confirm the user's own mute
   lists render. Tap one — confirm members list with avatars and
   names.
7. **Subscribe to a mute list.** From another user's profile or a
   mod list URL, subscribe to a mute list. Confirm it appears in
   the Mute lists section.
8. **Content labelers.** Open Content labelers. Confirm a list of
   subscribed labelers, each with name / description / a toggle
   to enable. Tapping one opens a sub-screen with the labeler's
   policies + per-label preferences (Show / Warn / Hide).
9. **Report a post.** From any post `…` menu → Report. Confirm a
   categorized picker appears (Spam / Sexual / Violent / Illegal /
   Other — RN parity). Confirm a labeler selection step (#0135).
   Submit. Confirm a success toast.
10. **Report a user.** From a profile's overflow → Report.
    Confirm the same categorized flow.
11. **Post interaction settings.** Open Settings → Post Interaction
    Settings (#0138). Confirm three sub-screens: Default reply
    rules, Default quote rules, Default DM rules. Each sub-screen
    has a Combine/Allow-list config matching RN's
    threadgate / postgate / DM settings.
12. **Content filters.** Confirm an adult-content toggle and a list
    of per-label preferences (Show / Warn / Hide) from each
    subscribed labeler.

## Sign-off checklist

- [ ] Moderation landing has all sections
