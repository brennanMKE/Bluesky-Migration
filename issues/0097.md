# 0097 — Composer is missing the self-labels (content warning) picker

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyComposer |
| **Platform** | All |
| **First seen** | 2026-05-06 |

## Description

Before posting, the React Native composer lets a user attach self-labels — content warnings that downstream clients use to blur or hide the post. The labels include `nudity`, `sexual`, `porn`, and `graphic-media` (this set follows the AT Proto self-label vocabulary). RN exposes them via a `LabelsBtn` in the composer toolbar that opens a dialog with toggleable rows (`Bluesky-ReactNative/src/view/com/composer/labels/`).

The SwiftUI composer has no UI for self-labels. A user cannot mark their own post as NSFW, which means SwiftUI-posted content is mis-categorized when viewed by users who hide adult content.

## Steps to reproduce

1. Open the SwiftUI composer.
2. Look for an icon/button to mark the post NSFW or as containing graphic media.
3. Observe there is none.

## Expected behavior

A label button (e.g., warning-triangle icon) in the composer toolbar that opens a sheet:

- Toggle: **Sexually explicit** (`porn`)
- Toggle: **Sexually suggestive** (`sexual`)
- Toggle: **Nudity** (`nudity`)
- Toggle: **Graphic media** (`graphic-media`)

Selected labels are written to `PostRecord.labels` as an `app.bsky.feed.post#selfLabels` object (see RN's post-create code path). The button shows a filled / colored state when any label is selected.

## Actual behavior

No labels button. `PostRecord` is created with no `labels` field regardless of content.

## Notes

- RN reference: `Bluesky-ReactNative/src/view/com/composer/labels/LabelsBtn.tsx` and `LabelsDialog.tsx`.
- Lexicon: `app.bsky.feed.post#selfLabels` (a `com.atproto.label.defs#selfLabels` value).
- File to edit: `BlueskyKit/Sources/BlueskyComposer/ComposerSheet.swift`; possibly add a `selfLabels` field to `PostRecord` in `BlueskyCore/Feed.swift` if not present.
- The four options above are the only valid self-label values for posts. Don't expose other values.
