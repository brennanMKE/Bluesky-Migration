# 0020 — Module 7 feature: inline post expansion not implemented

| | |
|---|---|
| **Status** | in-progress |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

In the React Native app, tapping a post in the thread view expands it inline to show full text, embeds, and the action bar without navigating to a new screen. The SwiftUI `ThreadView` does not have this behaviour.

## Steps to reproduce

1. Open any post thread with multiple replies.
2. Tap a collapsed post in the thread.

## Expected behavior

The post expands in-place to show its full content, embeds, and action bar.

## Actual behavior

Posts in `ThreadView` are always fully expanded or the tap navigates away — no inline expand/collapse toggle exists.

## Notes

Could be implemented as a `@State var expandedPostIDs: Set<String>` in `ThreadView` with conditional rendering of the full `PostCard` vs a compact row.
