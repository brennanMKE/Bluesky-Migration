## Description

Manual sign-off for the thread view — focal post styling, parent chain,
reply tree, reply/repost/quote/bookmark composer entry points, and the
focal-post stat row (Liked by / Reposted by / Quotes of).

## Test sequence

1. **Open a thread from feed.** Tap a post in the home feed. Confirm
   the thread loads with parent chain (if any) above and direct
   replies below.
2. **Focal post emphasis.** Confirm the focal post is visually
   distinct from parent and reply rows: larger avatar, larger text,
   full absolute date (e.g. "3:42 PM · Apr 28 2026"), and an
   expanded stat row with reply / repost / quote / like counts that
   are all tappable.
3. **Tap reply count.** Confirm scroll to the first reply (RN parity).
4. **Tap repost count → Reposted by.** Confirm `RepostedByScreen`
   pushes with the list of accounts that reposted.
5. **Tap quote count → Quotes of.** Confirm `QuotesOfScreen` pushes
   with all quote posts of the focal.
6. **Tap like count → Liked by.** Confirm `LikedByScreen` pushes
   with the list of accounts that liked.
7. **Parent chain navigation.** Tap a parent post above the focal.
   Confirm a new ThreadView pushes with that post as the focal.
8. **Reply tap → nested thread.** Tap a reply row. Confirm a new
   ThreadView pushes with that reply as the focal.
9. **Reply button → composer.** Tap the reply icon in the focal-post
   action bar. Confirm the composer sheet opens prefilled with
   `@author` and the reply-to context preview.
10. **Repost menu.** Tap the repost icon. Confirm a menu with two
    options: "Repost" and "Quote post". Tapping Quote opens the
    composer with the focal post embedded.
11. **Bookmark.** Tap the bookmark icon. Confirm the icon fills, and
    the post appears in Saved (next sign-in or pull-to-refresh in
    BookmarksScreen).
12. **Share / menu.** Tap `…`. Confirm: Copy link, Share, Open in
    browser, Translate, Mute thread, Mute user, Block user, Report
    post.
13. **Reply sort dropdown.** In a thread with multiple replies,
    confirm a sort dropdown is present (Hot / Newest / Oldest — RN
    parity) and changing the selection re-orders replies.
14. **Threadgate-hidden replies.** For a thread where the author has
    hidden specific replies, confirm hidden ones are shown beneath a
    "Show hidden replies" disclosure (matches RN). Confirm tapping
    expands them.
15. **Long-text expansion.** For a post over the truncation
    threshold, confirm "Show more" / "Show less" works (#0142).
16. **Video embed.** For a thread containing a video post, confirm
    inline playback or a fullscreen player works (#0147).

## Sign-off checklist

- [ ] Thread loads with parent chain + reply tree
- [ ] Focal post is visually emphasised (RN parity)
- [ ] All four stat-row taps push the correct destination
- [ ] Parent / reply taps push nested threads
- [ ] Reply / repost / quote / bookmark composer entry points all
      work and pre-fill correctly
- [ ] `…` menu contains the full RN set
- [ ] Reply sort dropdown present and functional
- [ ] Threadgate-hidden replies disclosure works
- [ ] Long-text expansion works
- [ ] Video embeds play (inline or fullscreen)
- [ ] No regressions in #0021, #0045, #0054, #0140, #0142, #0145, #0146, #0147
