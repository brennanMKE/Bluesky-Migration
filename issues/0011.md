## Description

## Steps to reproduce

1. Build and run the app in Xcode (simulator or device).
2. Sign in with a valid Bluesky account.
3. Force-quit the app.
4. Relaunch the app.
5. Observe whether the login screen is shown or the feed loads directly.

## Expected behavior

The app restores the previous session automatically and shows the home feed without requiring the user to log in again.

Not validated. May work correctly — `KeychainAccountStore` and `SessionManager.restoreLastSession()` are implemented — but no live gate run has been performed.

## Notes
