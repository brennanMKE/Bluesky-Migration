## Description

When multiple actors trigger the same notification group (e.g. five different people liked the same post), the React Native client displays up to 5 stacked actor avatars in the row, with a chevron (`v`) to expand the rest. The SwiftUI client currently shows only the first actor's avatar — losing the social context that "Point-Free **and 3 others** subscribed to this post" is meant to convey.

This is a follow-up to the iOS Notifications parity audit in #0069.

## Scope

### Avatar stack

- **Render up to 5 actor avatars** stacked horizontally in a row to the right of the bell icon.
- **Avatars overlap** by ~30% — leading edge of each subsequent avatar covers the trailing edge of the previous one (matches RN visual style).
- **Each avatar is tappable** — taps open that actor's profile via the standard navigation path.
- **Avatar sizes**: ~28pt diameter; adjust to feel right against the bell icon.
- **Order**: most-recent-first (matches the order of `actors` in the `GroupedNotification`).

### Expand chevron

- **Chevron button** (`chevron.down`) appears at the end of the avatar stack when the group has more than 5 actors.
- **Tap expands the row** to show all actors as a vertically-stacked list with full names.
- **Tapping the chevron again collapses** back to the 5-avatar stack.
- **Animate the expand/collapse** with a smooth height transition.

### Group label

- Update the group label to "**{first actor}** and **N others**" when the group has more than one actor — matches RN. The current SwiftUI label only shows the first actor.

## Implementation notes

- Code lives in `BlueskyKit/Sources/BlueskyNotifications/NotificationsScreen.swift` — `GroupedNotificationRow`.
- The `GroupedNotification` view-model already aggregates `actors: [ProfileBasic]`. The store doesn't need changes — only the row view.
- For overlap, use `ZStack` + `.offset` per avatar (or `HStack` with negative spacing).
- The chevron's visibility binds to `actors.count > 5`.

## Acceptance

- Notification rows with multiple actors show overlapping stacked avatars (up to 5).
- Tapping any avatar opens that actor's profile.
- A chevron button appears when there are more than 5 actors; tapping it expands to a full list.
- The group label uses the "X and N others" form when applicable.
- iOS Simulator and macOS builds pass.
