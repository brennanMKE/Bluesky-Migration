# 0005 — Repost button not functional on post cells

| | |
|---|---|
| **Status** | resolved |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The repost (recycling-arrows) button below each post in the feed has no effect when tapped or clicked.

## Steps to reproduce

1. Launch the app and sign in.
2. Navigate to any feed.
3. Tap or click the repost button on any post.

## Expected behavior

A confirmation popover or action sheet appears with options to Repost or Quote Post. Choosing Repost immediately reposted the item; choosing Quote Post opens the composer.

## Actual behavior

Nothing happens.

## Attachments

- `issues/0004/screenshot.png` — post action bar (shared screenshot)

## Notes

The repost action likely calls `app.bsky.feed.repost` via the AT Protocol client. Check that the button's action is wired up, the network call is implemented, and optimistic UI state (repost count increment, button highlight) is updated on success.
