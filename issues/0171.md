## Description

Manual sign-off for the curated-content surfaces — user lists, starter
packs, and bookmarks (Saved).

## Test sequence — Lists

1. **Open My Lists.** From Profile → `…` → My Lists. Confirm
   curation lists, mod lists, and starter packs are grouped on
   the landing screen (RN parity).
2. **List detail screen header.** Open a list. Confirm the header
   shows: avatar / banner if present, name, description, creator
   handle, member count, and action buttons (Follow list / Edit /
   Share) (#0125).
3. **List tabs.** Confirm Members / About tabs (#0126). Members
   shows profile rows; About shows the description + creator info
   + counts.
4. **Edit a list.** Tap the edit affordance. Confirm a sheet with
   name / description / banner / member-add / member-remove
   controls (#0127). Save. Confirm changes persist.
5. **Delete a list.** From the edit sheet or list menu, delete
   the list. Confirm a confirmation dialog, then the list is
   removed from My Lists (#0127).
6. **Create a list.** From My Lists, tap `+`. Confirm a Create
   sheet with name / description / privacy controls. Save.
   Confirm it appears in My Lists.

## Test sequence — Starter Packs

7. **Create a starter pack.** From My Lists or a dedicated
   Create entry, start a new starter pack. Confirm a multi-step
   wizard: (1) name + description, (2) pick people, (3) pick
   feeds, (4) review (#0128 — RN uses a wizard, not a single
   sheet).
8. **View a starter pack.** Open one of your own packs. Confirm
   the detail screen shows name, description, included people,
   included feeds, "Share" button.
9. **Follow all members.** Tap "Follow all". Confirm bulk follow
   succeeds.

## Test sequence — Bookmarks (Saved)

10. **Open Saved.** From the sidebar (macOS / iPad) or Profile →
    `…` → Saved (iPhone). Confirm the screen titled "Saved"
    appears (renamed from Bookmarks in #0060) and lists saved
    posts in card form matching the home feed.
11. **Pull to refresh.** Drag the list past top. Confirm refresh
    works.
12. **Tap a saved post.** Confirm navigation to its thread.
13. **Tap author on a saved post.** Confirm navigation to that
    profile.
14. **Unsave.** Tap the bookmark icon on a saved post's action
    bar. Confirm the row disappears immediately (optimistic).
15. **Empty state.** Unsave every post. Confirm a "Nothing saved
    yet" empty state appears (#0060).
