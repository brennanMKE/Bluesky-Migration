## Description

Manual sign-off for the Direct Messages tab — conversation list, message
threads, sending text and media, reactions, message requests inbox, and
group chats.

## Test sequence

1. **Open Messages tab.** Confirm the conversation list loads with
   one row per active thread, ordered newest-message-first.
2. **Conversation list row.** Each row shows: avatar, display name(s),
   last-message preview, relative timestamp, unread indicator if
   unread.
3. **Open a 1:1 chat.** Tap a conversation. Confirm the thread loads
   with messages ordered oldest-first at the top, newest at bottom,
   and the input bar at the bottom.
4. **Bubble layout.** Confirm sender bubbles are right-aligned with
   accent fill; receiver bubbles are left-aligned with neutral fill.
   Avatars are absent in 1:1 (matches RN); display name absent
   between consecutive messages from the same sender.
5. **Date dividers.** Between messages with more than ~5 min gap or
   across day boundaries, confirm a centered date divider
   ("Today", "Yesterday", "Mon May 6") appears (#0107).
6. **Per-message timestamp.** Long-press a bubble (or hover on
   macOS). Confirm the timestamp appears as a per-message detail.
7. **Send a text message.** Type and tap send. Confirm the message
   appears immediately (optimistic), the input clears, and a
   delivered checkmark appears once the server confirms.
8. **Send with mention / link / hashtag.** Type "@alice hey see
   https://example.com #cool". Confirm facets are detected and
   the message body renders with rich text on receipt (#0112).
9. **Send an image.** Tap the attachment / paperclip icon. Pick an
   image from the photo library. Confirm a preview appears before
   send, and the image renders as a thumbnail in the thread once
   sent (Module 11 / #0032).
10. **Reactions.** Long-press a received message. Confirm a reaction
    picker appears with common emoji. Pick one. Confirm the emoji
    appears under the bubble; tapping again removes it (#0110).
11. **Link card / post embed in a bubble.** If a message contains a
    Bluesky post URL, confirm the post renders as an inline embed
    inside the bubble (#0108).
12. **Scroll-to-bottom button.** Scroll up several screens worth.
    Confirm a small "↓ new messages" pill appears bottom-right.
    Tap it; confirm it scrolls to the latest message (#0109).
13. **System messages.** In a group chat, when a participant is
    added or removed, confirm a centered system message renders
    ("@alice added @bob") in the thread (#0111).
14. **Group chat.** Open a group conversation. Confirm sender info
    (avatar + name) above each message bubble (#0105).
15. **Message requests inbox.** Open the conversation list. Confirm
    a "Message requests" entry at the top with a count badge.
    Tap. Confirm requested conversations appear; accepting one
    moves it into the main list (#0032).
16. **Compose new chat.** Tap the new-message icon. Confirm a search
    field for picking recipients appears, and choosing one starts
    a new conversation.

## Sign-off checklist

- [ ] Conversation list loads and is correctly ordered
- [ ] 1:1 thread bubble layout matches RN
- [ ] Date dividers appear at right granularity
- [ ] Per-message timestamps on long-press
- [ ] Send text is optimistic and confirmed
- [ ] Facets detected and rendered in messages
- [ ] Image attachments work end-to-end
- [ ] Reactions: pick / remove
- [ ] Link cards / post embeds render in bubbles
- [ ] Scroll-to-bottom pill works
- [ ] System messages render
- [ ] Group chat sender info row present
- [ ] Message requests inbox accepts / rejects
- [ ] Compose-new-chat flow works
- [ ] No regressions in #0032, #0033, #0105–#0112
