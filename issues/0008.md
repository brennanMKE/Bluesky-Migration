# 0008 — Bookmark button missing from post action bar

| | |
|---|---|
| **Status** | in-progress |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The post action bar shows only four buttons (reply, repost, like, share). The bookmark button is absent entirely — it has not been added to the action bar layout.

## Steps to reproduce

1. Launch the app and sign in.
2. Navigate to any feed.
3. Inspect the row of buttons below any post.

## Expected behavior

A bookmark (bookmark ribbon) button is present in the action bar, allowing the user to save a post to their bookmarks list.

## Actual behavior

No bookmark button is visible. The action bar contains only four buttons.

## Attachments

- `issues/0004/screenshot.png` — post action bar showing four buttons, no bookmark

## Notes

Add a bookmark button to `PostActionBar` (or equivalent). Bluesky's AT Protocol does not have a native bookmark lexicon; bookmarks are typically implemented as a local list saved to `app.bsky.graph.list` or stored client-side. Confirm which approach the app is targeting. The button should toggle a filled/unfilled bookmark icon and update count/state accordingly.
