# 0006 — Like/Reaction button not functional on post cells

| | |
|---|---|
| **Status** | in-progress |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The like (heart) button below each post in the feed has no effect when tapped or clicked.

## Steps to reproduce

1. Launch the app and sign in.
2. Navigate to any feed.
3. Tap or click the heart/like button on any post.

## Expected behavior

The heart icon fills (or toggles back to unfilled), the like count increments or decrements optimistically, and `app.bsky.feed.like` (or `deleteLike`) is called against the AT Protocol API.

## Actual behavior

Nothing happens. Icon state and count do not change.

## Attachments

- `issues/0004/screenshot.png` — post action bar (shared screenshot)

## Notes

Verify the button action closure is connected. The like/unlike toggle requires tracking whether the current user has already liked the post (`viewer.like` URI on the post record). Check that the `PostViewModel` (or equivalent) holds this state and that the AT Protocol like/delete-like calls are implemented.
