1. Open Settings.
2. Look for a "Help" row.
3. Observe there is none.

## Expected behavior

Add a "Help" row to `SettingsScreen` between Languages and About sections. Tapping opens the help desk URL via `openURL`.

## Actual behavior

No Help row.
