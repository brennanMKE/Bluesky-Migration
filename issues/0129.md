# 0129 — Starter Pack screen missing included feeds, share button, and QR/deep-link affordances

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyLists |
| **Platform** | All |
| **First seen** | 2026-05-06 |

## Description

`StarterPackScreen` shows the header, member count, and a list of sample members. The React Native reference also displays:

- **Included feeds** — a section listing feed generators bundled in the pack with Pin / Add buttons.
- **Share** — a button that opens a system share sheet with the deep link (`https://bsky.app/starter-pack/<did>/<rkey>`).
- **QR code** — RN can render a QR for the deep link inside a sheet, useful for in-person sharing.
- **Copy link** — quick affordance for clipboard.

The "Follow All" button exists in SwiftUI, but the discoverability and shareability features are missing.

## Steps to reproduce

1. Open any starter pack.
2. Compare to RN.

## Expected behavior

Add to `StarterPackScreen`:

- **Included feeds** section between the header and member list, each row a `FeedCard` with a Pin button.
- **Share** button in the toolbar that presents a `ShareLink` with the deep-link URL.
- **QR code** option inside the share sheet (an image generated from the deep-link URL).
- **Copy link** in the share sheet.

## Actual behavior

Header + member list + Follow All only.

## Notes

- RN reference: `Bluesky-ReactNative/src/screens/StarterPack/StarterPackScreen.tsx`.
- QR generation: `CIQRCodeGenerator` filter on iOS, or one of the small Swift QR helpers.
- File to edit: `BlueskyKit/Sources/BlueskyLists/StarterPackScreen.swift`.
