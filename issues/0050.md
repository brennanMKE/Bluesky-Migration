## Description

Following the work in #0015, the component gallery views now have Light and Dark previews, but the majority of screens and views across all modules have no `#Preview` at all. These should each get a "— Light" and "— Dark" named preview using `.preferredColorScheme` at the preview root (and `.blueskyTheme(.dark)` where the Bluesky theme is used), following the pattern established in `dark-mode.md`.

## Notes
