# 0018 — Module 6 gate: home feed live validation (load, scroll, interactions)

| | |
|---|---|
| **Status** | open |

## Description

The Module 6 gate — confirm the home feed loads posts, infinite scroll works, pull-to-refresh works, and like/repost interactions persist across a refresh — has not been validated against the live API.

## Steps to reproduce

1. Sign in and navigate to the Home tab.
2. Scroll to the bottom of the loaded posts and confirm more posts load.
3. Pull down to refresh and confirm the feed reloads.
4. Like a post, refresh the feed, and confirm the like state is preserved.
5. Repost a post, refresh, and confirm the repost state is preserved.

## Expected behavior

All feed interactions work end-to-end with the live Bluesky API.

## Actual behavior

Not validated. See also issues 0004–0008 for known action bar bugs.
