# 0022 — Module 8 feature: profile Feeds and Lists tabs not implemented

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyProfile |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

`ProfileScreen` has Posts, Replies, Media, and Likes tabs but is missing the Feeds tab (feeds the user has created or pinned) and the Lists tab (lists the user has created).

## Steps to reproduce

1. Navigate to any profile.
2. Observe the available tabs in the segmented picker.

## Expected behavior

Two additional tabs — Feeds and Lists — appear alongside Posts/Replies/Media/Likes. Feeds tab shows the user's generated feeds via `app.bsky.feed.getActorFeeds`. Lists tab shows their lists via `app.bsky.graph.getLists`.

## Actual behavior

Only Posts, Replies, Media, Likes tabs exist. Feeds and Lists tabs are absent.

## Notes

`GetActorFeedsResponse` already exists in `BlueskyCore/FeedGenerator.swift`. `GetListsResponse` exists in `BlueskyCore/Graph.swift`. `ProfileViewModel` needs new `loadFeeds`/`loadLists` methods and `ProfileScreen` needs two new tab cases.
