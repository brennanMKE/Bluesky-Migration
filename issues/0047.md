# 0047 — Ellipsis (…) context menu missing from post action bar

| | |
|---|---|
| **Status** | resolved |
| **Module** | BlueskyFeed / BlueskyUI |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

Post cards in the feed and thread view have no ellipsis (…) button. On the Bluesky website and mobile app, tapping the three-dot button opens a context menu with actions including: delete post (own posts), copy post text, mute thread, mute words & tags, translate, pin to profile, and edit interaction settings.

## Steps to reproduce

1. Open the Home feed.
2. Look at the action bar on any post card.
3. Observe there is no ellipsis/more button.

## Expected behavior

Each post card has a `…` button at the trailing end of the action bar. Tapping it opens a menu with at minimum:
- **Delete post** (only shown for the viewer's own posts)
- **Copy post text**
- **Mute thread**

## Actual behavior

The action bar shows reply, repost, like, share, and bookmark — no ellipsis menu.

## Notes

`PostCard.Actions` already has extensible closure slots. A new `onMore: ((PostView) -> Void)?` action can be added and wired to a SwiftUI `Menu` button in the action bar. The delete post API call is `com.atproto.repo.deleteRecord` with collection `app.bsky.feed.post`.
