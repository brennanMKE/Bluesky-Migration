# 0024 — Module 8 feature: known followers chip not shown on profiles

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyProfile |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

When viewing another user's profile, the React Native app shows a "Followed by X, Y, and Z others you follow" chip below the stats row. This chip is absent from `ProfileHeaderView`.

## Steps to reproduce

1. Sign in and navigate to another user's profile (someone followed by accounts you also follow).
2. Look below the Following/Followers/Posts stats row.

## Expected behavior

A chip reads "Followed by [handle1], [handle2], and N others" linking to a list of known followers.

## Actual behavior

No known-followers chip is displayed.

## Notes

Requires a call to `app.bsky.graph.getKnownFollowers` (or equivalent). The endpoint should return profiles of mutual followers. Response type needs to be added to `BlueskyCore/Graph.swift` and called from `ProfileViewModel`.
