# 0017 — Module 6 feature: feed filter toggles (hide replies / hide reposts) not implemented

| | |
|---|---|
| **Status** | in-progress |
| **Module** | BlueskyFeed |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The home feed has no per-feed filter toggles to hide replies or hide reposts. In the React Native app these are per-feed preferences displayed in a filter menu above or within each feed.

## Steps to reproduce

1. Open the Home tab.
2. Look for a filter or options menu on the feed.

## Expected behavior

A filter control lets the user toggle "Hide replies" and "Hide reposts" per feed. The feed re-fetches with the appropriate `filter` parameter applied.

## Actual behavior

No filter controls exist. All replies and reposts are always shown.

## Notes

The AT Protocol `app.bsky.feed.getTimeline` and `app.bsky.feed.getFeed` endpoints accept a `filter` query parameter (e.g. `posts_no_replies`, `posts_with_replies`, `posts_with_media`). `FeedViewModel`/`FeedStore` would need a `filter: FeedFilter` state variable and UI to toggle it.
