## Description

Manual sign-off for the Search tab — typeahead, search results, hashtag
views, and the discovery / trending sections.

## Test sequence

1. **Open Search tab.** Confirm the search field is focused (or at
   least one tap away) and the discovery panel below shows trending
   topics + suggested users (matches RN).
2. **Typeahead — user.** Start typing a handle (e.g. `bsky.app`).
   Confirm a results dropdown appears within ~300ms with matching
   profiles. Confirm each row has avatar, display name, handle,
   and a follow indicator.
3. **Typeahead — post search.** Type a phrase. Confirm the dropdown
   shows both profile matches and a "Search posts for '…'" row at
   the bottom (RN parity).
4. **Submit search.** Tap the "Search posts for…" row or hit return.
   Confirm a full-screen results view appears with tabs: Top /
   Latest / People / Feeds (RN parity).
5. **Tab — Top.** Confirm post results render in the same card style
   as the home feed.
6. **Tab — Latest.** Confirm chronologically ordered post results.
7. **Tab — People.** Confirm profile rows with follow buttons.
8. **Tab — Feeds.** Confirm matching curated feed rows.
9. **Tap a post result → thread.** Confirm navigation works.
10. **Tap a user result → profile.** Confirm navigation works.
11. **Hashtag from feed.** From any post containing `#hashtag`, tap
    the tag. Confirm a hashtag screen opens with posts for that tag
    (#0027).
12. **Trending topics.** From the empty-search discovery panel,
    confirm a Trending Topics section with tappable items
    (#0026). Tap one → confirm it opens a topic or search view.
13. **Suggested users.** Confirm a "Suggested for you" list appears
    in the discovery panel with follow buttons.
14. **Recent searches.** Submit one search. Return to the empty
    search state. Confirm the recent search appears and is tappable.

## Sign-off checklist

- [ ] Search field is immediately usable
- [ ] Typeahead returns profiles within ~300ms
- [ ] Typeahead bottom row "Search posts for…" present
- [ ] Submit opens results view with 4 tabs (Top / Latest /
      People / Feeds)
- [ ] Each tab loads and renders correctly
- [ ] Result row taps navigate correctly
- [ ] Hashtag screen works
- [ ] Trending topics section renders
- [ ] Suggested users section renders
- [ ] Recent searches persist
- [ ] No regressions in #0026, #0027, #0028, #0051
