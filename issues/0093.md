# 0093 — Forgot-password recovery flow is missing

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyAuth |
| **Platform** | All |
| **First seen** | 2026-05-06 |

## Description

The React Native client lets a user reset a forgotten password directly inside the app: **`ForgotPasswordForm`** (enter email, request reset token), **`SetNewPasswordForm`** (enter token + new password) and **`PasswordUpdatedForm`** (confirmation). The entry point is a "Forgot?" link inline with the password field in `LoginForm.tsx` (lines 256–266).

The SwiftUI `LoginView` has no "Forgot password?" link and no recovery screens at all. A user who has forgotten their password is locked out unless they leave the app and reset on bsky.app.

## Steps to reproduce

1. Launch the SwiftUI app on the login screen.
2. Look for a "Forgot password?" / "Forgot?" link near the password field.
3. Observe there is none.

## Expected behavior

- A "Forgot?" link to the right of the password field's label (or below the password field), matching RN's placement.
- Tapping it opens the recovery flow:
  1. **Request screen**: email field + hosting provider picker → calls `com.atproto.server.requestPasswordReset`.
  2. **Set new password screen**: token field (the user pastes the code from the email) + new-password field with confirmation → calls `com.atproto.server.resetPassword`.
  3. **Confirmation screen**: "Password updated" message + "Sign in now" CTA returning to `LoginView` with the email pre-filled.
- Each screen exposes inline error display for invalid tokens / weak passwords / unknown emails.

## Actual behavior

No "Forgot password?" link on `LoginView`; no recovery screens exist anywhere in `BlueskyAuth`.

## Notes

- RN reference: `Bluesky-ReactNative/src/screens/Login/ForgotPasswordForm.tsx`, `SetNewPasswordForm.tsx`, `PasswordUpdatedForm.tsx`. `index.tsx` orchestrates the form transitions.
- AT Proto endpoints needed: `com.atproto.server.requestPasswordReset`, `com.atproto.server.resetPassword`. Lexicons may not exist yet in `BlueskyCore/Auth.swift`.
- File to edit: `BlueskyKit/Sources/BlueskyAuth/LoginView.swift` (add the link + sheet/navigation), and create new `ForgotPasswordView.swift` / `ResetPasswordView.swift`.
