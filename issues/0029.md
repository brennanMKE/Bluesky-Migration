# 0029 — Module 10 feature: grouped notifications not implemented

| | |

| **Status** | implemented |

| **Platform** | All |

## Description

The Notifications screen shows each notification as an individual row. The React Native app groups related notifications (e.g. "3 people liked your post") into a single collapsed row.

## Steps to reproduce

1. Open the Notifications tab on an account with multiple likes on a single post.
2. Observe whether likes are grouped or listed individually.

## Expected behavior

Multiple notifications of the same type for the same subject are collapsed into a single grouped row (e.g. "Alice, Bob, and 5 others liked your post").

## Actual behavior

Each notification is shown as a separate row regardless of type or subject.

## Notes

The grouping logic would live in `NotificationsStore` or a new helper that groups `[NotificationView]` by `reasonSubject` + `reason`. The `NotificationsScreen` row renderer needs a grouped variant.
