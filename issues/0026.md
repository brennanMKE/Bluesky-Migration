# 0026 — Module 9 feature: trending topics section not implemented

| | |
|---|---|
| **Status** | resolved |
| **Module** | BlueskySearch |
| **Platform** | All |
| **First seen** | 2026-04-29 |

## Description

The Search screen is missing the trending topics section that appears in the React Native app when the search bar is empty. Currently only suggested accounts are shown.

## Steps to reproduce

1. Open the Search tab with an empty search bar.
2. Observe the content below the search bar.

## Expected behavior

A "Trending" section appears showing trending hashtags or topics, fetched from the Bluesky API.

## Actual behavior

Only a "Suggested Accounts" section is shown. No trending topics appear.

## Notes

The API endpoint for trending topics may be `app.bsky.unspecced.getTrendingTopics` or similar. Response type would need to be added to `BlueskyCore/Search.swift` and `SearchViewModel`/`SearchStore` extended with a `loadTrending()` method.
