# 0009 — Post action buttons lack tooltips on macOS

| | |
|---|---|
| **Status** | open |
| **Module** | BlueskyFeed |
| **Platform** | macOS |
| **First seen** | 2026-04-29 |

## Description

Hovering the cursor over the post action buttons (reply, repost, like, share, bookmark) shows no tooltip. On macOS, users expect a tooltip to appear after a short hover delay identifying the button's action.

## Steps to reproduce

1. Launch the app on macOS and sign in.
2. Navigate to any feed.
3. Hover the cursor over any of the action buttons below a post and wait ~1 second.

## Expected behavior

A tooltip appears with a short label: "Reply", "Repost", "Like", "Share", or "Bookmark".

## Actual behavior

No tooltip appears.

## Notes

In SwiftUI, add `.help("Reply")` (and equivalent) to each button in the post action bar. `.help()` renders as a tooltip on macOS and as an accessibility label on iOS, making it a free accessibility improvement at the same time.
