## Description

The React Native reference shows an inline composer prompt at the very top of the Home feed, between the tab strip and the first post. It contains the user's avatar on the left, the prompt text "What's up?" centered, and a camera icon plus a photo-library icon on the right. Tapping anywhere in the row opens the full `ComposerSheet`. The SwiftUI iOS app has no equivalent — the only composer entry is the top-right pencil icon (replaced by a FAB in #0071).

Adding this prompt closes a discoverability gap and matches the long-standing RN pattern.

## Scope

- **Add a new view** at the top of the Home feed list, below the tab strip and above the first post.
- **Layout**:
  - Leading: circular avatar (24pt) for the signed-in user
  - Center: secondary-color text "What's up?" left-aligned with a small leading gap from the avatar
  - Trailing: two outlined icon buttons — camera (`camera`) and photos (`photo`) — separated by a small gap
- **Tap anywhere on the row → open `ComposerSheet`** with no pre-set body, no reply target, no quoted post.
- **Tap camera button → open `ComposerSheet`** with the camera-capture flow primed (iOS only — uses `UIImagePickerController` with `.camera` source).
- **Tap photos button → open `ComposerSheet`** with the photo-picker flow primed.
- **Visible only on Home**, not on other feeds. Hide when scrolled past the top? Optional polish — match RN behavior (RN keeps it pinned at top of the feed list, scrolls away normally).

## Implementation notes

- New `InlineComposerPrompt` view in `BlueskyFeed` (or `BlueskyComposer`).
- Inject as the first row of the `LazyVStack` in `FeedView`, gated `#if os(iOS)`.
- Pre-priming the composer with camera/photos: extend `ComposerSheet` to take an optional initial-attachment-source enum (`.none`, `.camera`, `.photoLibrary`).
- On macOS, do not render this prompt — the macOS toolbar already provides a compose action and the layout doesn't fit.

## Acceptance

- iOS Home: an inline composer row with avatar, "What's up?", camera, and photo icons appears at the top of the feed.
- Tapping the row, the camera button, or the photo button opens `ComposerSheet` with the appropriate primed state.
- macOS Home: unchanged.
- iOS Simulator and macOS builds pass.
