## Live testing observations (2026-04-30)

**Messages tab fails to load** — the conversation list shows an error banner: "The operation couldn't be completed. (BlueskyCore.ATError error 2.)"

**Root cause identified from logs:**
```
convos fetch error: BlueskyCore.ATError.xrpc(code: "MethodNotImplemented", message: "Method Not Implemented")
request convos fetch error: BlueskyCore.ATError.xrpc(code: "MethodNotImplemented", message: "Method Not Implemented")
```

The `chat.bsky.convo.listConvos` calls are being sent to `bsky.social`, but the Bluesky chat service runs on a separate proxy endpoint (`https://api.bsky.chat`). All `chat.bsky.*` lexicons must be routed to this endpoint instead of the standard PDS. The `ATProtoClient` needs to detect `chat.bsky.*` lexicons and use the chat service URL.
